Aggiungere record

Ciascun documento di Conto corrisponde a un anno di transazioni, da gennaio a dicembre. Per aggiungere un record basta scegliere il mese appropriato dal menú a tendina in alto a sinistra, compilare i campi di testo e premere il tasto Return o il pulsante Aggiungi. La data deve rispettare il formato specificato nelle Preferenze. Se il campo Data: è lasciato vuoto, Conto inserisce automaticamente la data corrente.

Ci sono due modi per decidere se l'importo inserito è un'entrata o una spesa: si può selezionare l'appropriato tipo di importo nelle caselle in basso a sinistra nella finestra principale, oppure è possibile inserire un valore negativo. Un'entrata negativa è considerata una spesa, e viceversa. Se il campo Importo: è lasciato vuoto, Conto inserisce zero. Se si vogliono inserire sia un'entrata sia un'uscita nella stessa riga, si veda come modificare i record. Se la casella Spunta è selezionata, un segno di spunta appare nei record inseriti.
