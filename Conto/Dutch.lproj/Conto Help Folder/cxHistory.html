Version history

1.2.0

  • Added a search field à la iTunes. As you type, Conto displays only the records whose descriptions contain the text you entered. At the same, total income, expense and balance are computed on the fly over the shown items.
  • When viewing help in the Help Viewer, now links should always work.
  • After entering a record, input fields are cleared.

1.1.2

  • Fixed a bug which prevented cut/copy/paste and drag and drop from working properly in all circumstances. Note: copying/dragging from documents created with versions less than 1.1.2 may not work!

1.1.1

  • Fixed a major bug which, in some cases, caused amounts in records not to be displayed when opening again a document saved without encryption (thanks to Stijn van Nuffelen for pointing this out).
  • Added French localization (everything but help), thanks to Vincent Zorzi.

1.1.0

  • Added predefined descriptions: now it is possible to define descriptions and associate an operation for the income field and an operation for the expense field to each description.
  • The user can (de)activate autocompletion of descriptions.
  • The application now remembers columns' width and order and windows' size and position.
  • Added a "monthly balance" field in the main window.
  • Help rewritten (now it uses the Help Viewer).
  • Added buttons to move to previous/next month.
  • Implemented Cut/Copy/Paste (previous Copy is now "Copy as Text").
  • Implemented Drag and Drop functionalities: now you can sort account data and informations by dragging them in the desired position. You can copy records from one document to another by simply dragging them. You can reorder predefined descriptions in the Preferences window by drag and drop.
  • The user can now (de)activate a grid.
  • The font in the Account Info drawer changes according to Preferences.
  • Preferences window now contains a TabView.
  • (Almost complete) localization in Dutch (thanks to Marco van Weel).
  • Minor code optimizations.

1.0.1

  • Added application and document icons
  • Localized in Italian

1.0.0

  • First released version
